Application Uptime Monitoring

Track web app,lication availability and performance using Prometheus, Blackbox Exporter, and Grafana.

📈 Project Overview

This project monitors website uptime and response time using the Prometheus monitoring system and the Blackbox Exporter. Grafana is used to visualize the metrics on a customizable dashboard.

🛠 Features

  • 🔍 Monitor website status (Up/Down) using Blackbox Exporter
  • ⏱ Track response time and HTTP status codes
  • 📊 Grafana dashboards for real-time visualization
  • 🐳 Dockerized setup with Terraform provisioning

🧱 Architecture

The setup includes three main services connected via Docker:

  1. Prometheus - scrapes metrics from exporters
  2. Blackbox Exporter - probes HTTP endpoints
  3. Grafana - visualizes the data with dashboards
